# Puma Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing sales performance using a structured dataset and an interactive Power BI dashboard.

The goal is to transform raw sales data into clear insights that help understand:
- Revenue and profit trends
- Product performance
- Regional contribution
- Customer behavior

The dashboard is designed using a star schema data model and includes five analytical pages.

---

## Dashboard Pages

### 1. Executive Overview
- Summary of key metrics such as revenue, profit, and orders
- Monthly revenue trends
- Revenue distribution by region and product line

### 2. Sales Performance Analysis
- Revenue and profit comparison over time
- Channel wise performance
- Units sold by product categories

### 3. Product Analysis
- Product level revenue and profit
- Cost vs MRP comparison
- Profit margin by product line

### 4. Region Analysis
- Revenue and profit by region
- Order distribution across cities
- Regional performance comparison

### 5. Customer Analysis
- Revenue by gender
- Product preference by customer segment
- Size wise sales distribution

## How to Use

1. Open the .pbix file
2. Use slicers to filter data by:
   - Region
   - Product Line
   - Gender
   - Size
   - Date
3. Navigate between dashboard pages
4. Click visuals for cross filtering
5. Use drill down in time based charts
