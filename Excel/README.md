Retail Sales Analysis Dashboard

Project Overview
Retail sales analysis project built in Microsoft Excel. The dashboard analyzes sales performance, product categories, regional sales distribution, and profit margins.

Dataset
315 retail transaction records with columns for order date, region, category, quantity, discount, sales, and profit.

Key Analysis
Total sales performance
Profit and profit margin tracking
Regional sales comparison
Product category performance
Return status analysis

Tools
Microsoft Excel
Pivot Tables
Data Cleaning
Dashboard Visualization

Project Output
Interactive Excel dashboard summarizing key retail business metrics.
